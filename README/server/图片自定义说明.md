# 图片自定义说明

## 概述

系统支持用户自定义取票页面和显示屏的背景图片。有两种方式可以自定义图片：

1. **外部图片目录**（推荐）- 将图片放在可执行文件同目录的 `pic` 文件夹中
2. **通过管理员后台设置** - 在系统设置中修改图片路径

## 方式一：外部图片目录（推荐）

### 打包后的可执行文件

当程序被打包成可执行文件（.exe）后：

1. **图片存放位置**：在可执行文件（`queue-system.exe`）同目录下创建 `pic` 文件夹
2. **支持的图片文件**：
   - `ticket_bg.jpg` - 取票页面背景图片
   - `display_bg.png` - 显示屏背景图片

### 目录结构示例

```
C:\QueueSystem\
├── queue-system.exe      # 可执行文件
├── pic\                  # 图片文件夹（需要手动创建）
│   ├── ticket_bg.jpg     # 取票页面背景图片
│   ├── display_bg.png    # 显示屏背景图片
│   └── README.txt        # 说明文件
└── data\                 # 数据库文件夹（自动创建）
```

### 使用步骤

1. 在可执行文件同目录下创建 `pic` 文件夹（如果不存在）
2. 将您的图片文件放入 `pic` 文件夹中
3. 确保文件名与上述文件名一致
4. 重启应用程序即可生效

### 图片要求

- **格式**：JPG、PNG、GIF 等 Web 浏览器支持的格式
- **建议尺寸**：
  - `ticket_bg.jpg`: 宽度 1920px 或以上
  - `display_bg.png`: 宽度 1920px 或以上
- **文件大小**：建议不超过 5MB

### 优先级

- 如果 `pic` 文件夹中存在对应的图片文件，程序会优先使用外部图片
- 如果外部图片不存在，程序会使用打包内的默认图片

## 方式二：通过管理员后台设置

### 设置步骤

1. 登录管理员后台（`http://[服务器IP]/admin`）
2. 进入"系统设置"页面
3. 找到以下设置项：
   - `ticket_banner_image` - 取票页面背景图片路径
   - `display_banner_image` - 显示屏背景图片路径
4. 修改图片路径（例如：`/pic/ticket_bg.jpg`）
5. 保存设置

### 路径说明

- 图片路径应该是相对于 `public` 目录的路径
- 例如：`/pic/ticket_bg.jpg` 对应 `public/pic/ticket_bg.jpg`
- 如果使用外部图片目录，路径仍然是 `/pic/ticket_bg.jpg`（程序会自动查找外部目录）

## 开发环境

在开发环境中，图片文件位于：
- `queueSystem-client/public/pic/`

您可以直接替换这些文件来测试自定义图片效果。

## 注意事项

1. **文件命名**：必须使用正确的文件名（区分大小写）
2. **文件路径**：确保图片文件在正确的位置
3. **重启应用**：修改图片后需要重启应用程序才能生效
4. **图片格式**：建议使用 JPG（取票页面）和 PNG（显示屏）格式
5. **图片大小**：过大的图片可能影响加载速度

## 故障排除

### 图片不显示

1. 检查文件是否存在且文件名正确
2. 检查文件路径是否正确
3. 检查图片格式是否支持
4. 查看浏览器控制台是否有错误信息
5. 重启应用程序

### 图片显示不正确

1. 检查图片尺寸是否合适
2. 检查图片格式是否正确
3. 尝试使用不同的图片文件测试

## 技术支持

如有问题，请检查：
- 应用程序日志
- 浏览器控制台错误信息
- 图片文件是否损坏
